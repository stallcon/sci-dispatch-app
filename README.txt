SCI Dispatch PWA Package

Run locally:
1. Open a terminal in this folder.
2. Run: python -m http.server 8000
3. Open http://localhost:8000

Install on phone:
1. Host these files on any web host or run them locally on your network.
2. Open the site in Safari (iPhone) or Chrome (Android).
3. Use Add to Home Screen / Install App.

Notes:
- Live weather lookup uses Open-Meteo and Nominatim in the browser.
- Employee login and notifications are simulated in this PWA package.
